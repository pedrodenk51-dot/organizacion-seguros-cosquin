# Organización Seguros Cosquín

Sitio web institucional de Organización Seguros Cosquín, oficina de productores asesores de seguros en Cosquín, Córdoba.

## Descripción

Este proyecto es una landing page/web institucional pensada para mostrar:

- trayectoria de la organización
- equipo de productores matriculados
- compañías con las que trabajamos
- coberturas principales
- formulario de contacto con envío por WhatsApp
